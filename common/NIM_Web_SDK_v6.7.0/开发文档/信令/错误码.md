# <span id="错误码">错误码</span>

| 错误码 |   说明 |
| :--: | :------ |
| 10201  | 执行的操作已经成功，只是对方不在线（推送可达，但是离线） |
| 10202  | 执行的操作已经成功，只是对方推送不可达 |
| 10404 | 对应的频道不存在 |
| 10405 | 对应的频道已存在                                         |
| 10406 | 不在频道内 |
| 10407  | 已经在频道内 |
| 10408 | 邀请不存在或已过期 |
| 10409 | 邀请已经拒绝 |
| 10410 | 邀请已经接受了 |
| 10417 | 加入频道uid 冲突 |
| 10419 | 频道人数超限 |
| 10420 | 已经在频道内（自己的其他端） |
 
  **提示：10201 、10202 比较特殊，使用时关注一下**
