聊天室成员类型

聊天室成员分为固定成员和游客两种。固定成员又分为房主、管理员、普通成员和受限成员四种。禁言用户和拉黑用户都属于受限用户。

  • 'owner' (房主)
  • 'manager' (管理员)
  • 'restricted' (受限制, 被拉黑或者禁言)
  • 'unset' (未设置)
  • 'common' (普通成员)
  • 'guest' (游客)
  • 'anonymous' (匿名非注册用户，非云信注册用户)
